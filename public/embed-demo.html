Iframe Embedding

Simple HTML iframe for basic integration. Perfect for quick setup and testing.

<iframe src="https://your-frontend-domain.com/chat-embed?chatbotId=YOUR_CHATBOT_ID&domain=your-website.com" style=" position: fixed; bottom: 20px; right: 20px; width: 380px; height: 600px; border: none; border-radius: 16px; box-shadow: 0 10px 40px rgba(0,0,0,0.15); z-index: 9999; background: transparent; " allow="microphone" title="RankVed Chatbot"> </iframe>
